Astro 负责静态构建，MDX 让文章可以直接嵌入组件，Docker 则把最终站点变成一个可复制部署的镜像。

## 为什么使用 MDX

普通 Markdown 很适合写作，但当页面需要交互 demo、视频、图表或局部组件时，MDX 会更合适。

## 部署模型

生产环境不运行 Node 服务。Docker 构建阶段生成 `dist/`，Pagefind 建立静态搜索索引，最终由 Nginx 托管。
