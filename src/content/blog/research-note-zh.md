研究笔记适合保存为普通文件。它们可以被 Git 版本管理、在本地搜索、跨系统迁移，也不依赖后台 CMS。

这个网站把内容层和部署层分开。即使以后部署环境变化，正文仍然可以保留在 Markdown 和 MDX 中。
